# 🚀 AprendendoComBecrownd

Bem-vindo ao meu repositório **AprendendoComBecrownd**! 👩‍💻✨
Aqui estou registrando minha jornada de aprendizado em programação através dos exercícios da plataforma **Beecrowd**.
O objetivo deste repositório é acompanhar minha evolução, praticar lógica de programação e melhorar minhas habilidades em desenvolvimento.

## 📚 Sobre o projeto

Este repositório contém soluções de exercícios resolvidos no Beecrowd, organizados conforme vou aprendendo novos conceitos.
Ainda estou desenvolvendo e atualizando constantemente este projeto, então novos exercícios serão adicionados ao longo do tempo 🚀

## 🛠️ Tecnologias utilizadas

* JavaScript
* Node.js
* Lógica de Programação
* Algoritmos

## 🎯 Objetivos

* Melhorar minha lógica de programação
* Aprender resolução de problemas

* Evoluir diariamente na área de tecnologia

## 📈 Minha evolução

Mesmo os desafios mais difíceis fazem parte do aprendizado!
